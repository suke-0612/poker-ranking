# ポーカーランキングアプリ 管理者向けマニュアル

このドキュメントは、ポーカー大会の進行およびスコア管理を行う「管理者（Admin）」向けの使い方をまとめた取扱説明書です。

---

## 🧭 運用の基本的な流れ（ワークフロー）

大会を新しく開催し、進行していくための基本的な手順は以下の通りです。

1. **大会の作成** (`大会管理`)：まずは新しい大会（Tournament）を開始します。
2. **初期設定** (`マスタデータ管理`)：参加するプレイヤーの登録と、順位（1〜10位）ごとの景品を設定します。
3. **ゲーム進行とスコア入力** (`スコア入力`)：ゲームが1回終わるごとに「新しいゲーム（ラウンド）」を追加し、変動したスコアを更新します。
4. **ランキング表示** (`トップページ`)：参加者や観客が見るための画面です。自動で15秒ごとに最新のスコアに更新されます。

---

## 🖥️ 各画面の詳細な使い方

### 1. 大会管理（Tournaments）
**URLパス:** `/admin/tournaments`

大会全体のライフサイクル（開始・終了）を管理する画面です。

*   **最初の大会を開始する**
    初期状態では大会が存在しないため、まずは画面内の赤いボタン「**現在の大会を終了して新しく開始**」をクリックしてください。これで第1回の大会が「開催中（Active）」となります。
*   **大会をリセットする（次回の大会を始める）**
    現在の大会がすべて終了し、新しく別の大会をゼロからスタートしたい場合も、同じボタンをクリックします。
    ※ 過去の大会のスコアデータは消去されず、「終了済み（Archived）」として履歴に保存されます。

### 2. マスタデータ管理（Master Data）
**URLパス:** `/admin/master`

プレイヤー情報と、開催中の大会の景品情報を管理する画面です。

*   **プレイヤーの登録と管理**
    *   左側の「プレイヤー」セクションで、新しく参加するプレイヤーの名前を入力し「追加」ボタンを押します。
    *   途中退出したプレイヤーなどは「有効化 / 無効化」ボタンでステータスを切り替えることで、スコア入力画面やランキング画面から一時的に非表示にすることができます。
*   **景品（Prizes）の設定**
    *   右側のセクションで、現在開催中の大会における「1位〜10位」までの景品名を入力し、「景品を保存」をクリックします。（未入力の順位は画面にハイフンで表示されます）

### 3. スコア入力（Scores）
**URLパス:** `/admin/scores`

毎回のゲーム終了後に、プレイヤーの持ち点（スコア）を記録・更新するメイン画面です。

*   **新しいゲーム（ラウンド）の開始**
    *   ゲームが1回終わるごとに、右上の「**新しいゲームを開始**」ボタンをクリックします。
    *   **【重要】** このボタンを押すと、自動的に「直前のゲームの全員のスコア」がそのまま新しいゲームにコピーされます。そのため、全員のスコアを毎回ゼロから入力し直す必要はありません。
*   **スコアの更新**
    *   リストの中から、前回のゲームから**得点が変動したプレイヤーの点数のみを書き換え**ます（入力は絶対値＝現在の最終的な持ち点）。
    *   入力が終わったら、画面右下の「**このゲームのスコアを保存**」ボタンを必ず押してください。
*   **過去のゲームのスコア修正**
    *   「ゲーム（ラウンド）の選択」タブ（Game 1, Game 2...）をクリックすると、過去のラウンドのスコアに戻って数値を修正することができます。

### 4. ライブランキング画面（View Live Ranking）
**URLパス:** `/` （トップページ）

会場のモニター等に映し出すための閲覧専用画面です。

*   **自動更新（ポーリング）**
    *   管理者がスコアを保存すると、最大15秒以内にこの画面に自動的に最新ランキングが反映されます。手動でブラウザをリロードする必要はありません。
*   **同点（タイ）の処理**
    *   複数のプレイヤーが全く同じスコアになった場合、システムは「マスタデータに早く登録された順（古参優先）」で順位を決定します。

---

## 🔒 ログイン（認証）について

当システムはセキュリティのため「Basic認証」によるアクセス制限がかかっています。
アクセス時にダイアログが表示された場合は、システム管理責任者から共有された**ID（ユーザー名）とパスワード**を入力してログインしてください。
